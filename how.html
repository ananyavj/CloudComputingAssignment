How TimeWeave Works

Scheduling across timezones is a nightmare. This tool makes it a visual breeze. Here's how to master it in 30 seconds.

1

Add your team members

Type a name and select their specific city or timezone. Don't worry about UTC offsets; we handle daylight savings and everything automatically using native browser intelligence.

2

Set their working hours

Define when each person is usually "at their desk" in their own local time (e.g., 9 AM to 5 PM). You can add multiple work shifts if they take mid-day breaks.

3

Read the Vintage Grid

The 24-hour grid shows everyone's relative availability in the vintage professional color system.

• Sage Green = Member is working
• Vintage Blue = Unanimous meeting intersection! 🌏
• Dusty Rose ✨ = The currently selected meeting window
4

Cycle through "Best Windows"

If there are multiple times everyone is free, use the "Next Window →" button in the bottom summary bar to cycle through options. Adjust the "Meeting Duration" in settings to find longer or shorter blocks.
